## 👋 About Me

Hey! I'm **Tucker Johnson** — a frontend software developer at **[Collabetive](https://www.thecollabetive.com/)**, where I work with React, Mantine UI, and TypeScript.

Alongside my day job I'm completing a **Master of Science in Software Engineering** at the **University of St. Thomas**, deepening my expertise in cloud architecture, databases, and software design.

My sweet spot is where clean, purposeful UI meets real-world utility — whether that's geo-mapping data, solving business problems, or experimenting with the latest AI tooling.

**What gets me excited:**
- 🗺️ **Geo & mapping projects** — turning open datasets into interactive, searchable map experiences
- 💼 **Business-first software** — tools that save time and deliver measurable value
- 🎨 **Exceptional UX** — obsessing over interfaces that feel intuitive and effortless
- 🤖 **AI-assisted development** — integrating LLMs into my workflow and exploring prompt engineering
- ☁️ **Cloud & DevOps** — Azure, AWS, Docker, CI/CD pipelines, and infrastructure as code

> 🌲 **Currently building:** [Explore More MN](https://github.com/Tuck1297/mn-map-repo) — an interactive web app consolidating Minnesota's outdoor recreation data (state parks, campgrounds, trails) into one searchable, map-powered platform.
